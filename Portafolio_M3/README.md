# Portafolio M3 – E-commerce CLI (Python)

Aplicación de consola que modela un e-commerce básico usando contenidos del Módulo 3:
tipos de datos, condicionales, ciclos, estructuras de datos y funciones.

## Ejecución
```bash
python ecommerce_m3.py
```

## Funcionalidades (MVP)
- Catálogo mínimo de 5 productos (id, nombre, categoría, precio).
- Menú principal en ciclo `while` hasta opción **0) Salir**.
- Búsqueda por nombre o categoría.
- Carrito en memoria (agregar por id + cantidad con validaciones).
- Visualización de carrito con subtotales y total a pagar.
- Vaciar carrito.

## Evidencia para revisión
- `ecommerce_m3.py` contiene:
  - funciones separadas (menú, catálogo, búsqueda, carrito, totales)
  - validación de id y cantidad
  - mensajes claros al usuario
