# ecommerce-frontend-m2 (MVP)

Frontend de e-commerce (Módulo 2) construido con **HTML5 semántico**, **Bootstrap (CDN)** y **JavaScript**.

## Requisitos cubiertos (pauta)
- Home con grilla de productos (Bootstrap Cards).
- Detalle de producto (imagen, precio, descripción, botón Agregar).
- Carrito simulado con contador en navbar (localStorage).
- Responsive (mobile-first).

## Cómo ejecutar
Opción 1 (rápida): abrir `index.html` con doble clic.

Opción 2 (recomendada): levantar un servidor local (evita restricciones de módulos ES):
- VSCode → extensión **Live Server** → "Open with Live Server"
- o Python:
  - `python -m http.server 8000`
  - abrir `http://localhost:8000`

## Estructura
- `index.html`, `product.html`, `cart.html`, `contact.html`
- `assets/js/` lógica JS
- `assets/css/` estilos mínimos
